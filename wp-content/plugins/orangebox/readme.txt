== Description ==

OrangeBox is a Lightweight, Cross-Browser, Automated jQuery Lightbox Script. It can display images, quicktime videos, YouTube and Vimeo videos, Flash SWF files, and inline content (along with links in the inline content to open another lightbox). It also groups items together (regardless of their content type) and shows easy to use navigation.

This is the initial release of the OrangeBox Wordpress Plugin.
The Plugin runs on OrangeBox 2.0.3

= What's new in OrangeBox 2.0.3 =

* [AddThis](http://www.addthis.com/) widget added for content sharing
* Option to enable/disable AddThis
* New Attribute to set shared URL for AddThis
* New Attribute to disable AddThis per link

[See the full change log](http://orangebox.davidpaulhamilton.net/)


== Installation ==

To install the plugin, please follow these steps

1. Upload the `orangebox` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Go to the OrangeBox Options/Settings page in your WordPress Dashboard and set up your options
1. For more information on usage and customization go to the http://orangebox.davidpaulhamilton.net

= How To: =
The plugin automatically adds OrangeBox functionality to image/video links. There is no shortcode and no extra steps. Read the [full documentation](http://orangebox.davidpaulhamilton.net/docs.html) for more information 

= API & Customization =
OrangeBox can be fully customized through the OrangeBox settings page. Visit this page for information about [API and Customization](http://orangebox.davidpaulhamilton.net/api.html)

== Frequently Asked Questions ==

= How do I use OrangeBox with NextGen Gallery? =

To setup NextGen Gallery to use OrangeBox:
1. Click on "Options" under "Gallery" in your WordPress Dashboard (on the left)
1. Select the "Effects" tab
1. Select "Lightbox" as the JavaScript Thumbnail effect
